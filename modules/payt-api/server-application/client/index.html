LIFE PAYT API documentation version 1

http://api.life-payt.eu/api

Welcome

Welcome to the LIFE PAYT API Documentation. In order to use it propeprly replace location with your location name in the request string. E.G: 'aveiro-container' or 'condeixa-container'. Otherwise it will not work and you will get a 'Not Authorized' Message.

Users (API)

Endpoint for user login that returns the access token.

/Users/login post

Returns ID. To get your identifier call /Users/login with your credentials in a object with username and password.

post

This method returns upon correct credentials the access token of the user necessary for procteted endpoints.

post /Users/login

This method returns upon correct credentials the access token of the user necessary for procteted endpoints.

  • Request
  • Response

Query Parameters

  • credentials: required(json)

    Example:

    {
  "username": "testuser",
  "password": "00000"
}

Body

Media type: example

Type:

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' -d 
'{ \ 
   "username": "testuser", \ 
   "password": "000000" \ 
 }' 
 'https://api.life-payt.eu/api/Users/login'

HTTP status code 200

Copy the "id" field for using as access token for further actions.

Body

Media type: application/json

Type:

{
  "id": String,
  "ttl": Number,
  "created": Date,
  "userId": String
}

Example:

{
  "id": "1234567890",
  "ttl": 10000,
  "created": "1977-07-10T14:14:27.914Z",
  "userId": "0987654321"
}
